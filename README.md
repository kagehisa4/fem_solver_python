\# 🧮 FEM Solver (Python)

# Note that this solver works only for torsion mode shapes for any cantilever beam. 

A Python-based \*\*Finite Element Method (FEM) solver\*\* developed for beam modal analysis.  

This project demonstrates numerical methods, matrix assembly, and eigenvalue-based analysis using scientific computing tools.



\---



\## 🚀 Features



\- 📐 Stiffness and mass matrix assembly  

\- 🔧 Boundary condition handling  

\- 📊 Eigenvalue problem solving for natural frequencies  

\- ⚡ Efficient implementation using NumPy and SciPy  



\---



\## 🛠 Tech Stack



\- \*\*Language:\*\* Python  

\- \*\*Libraries:\*\* NumPy, SciPy  

\- \*\*Concepts:\*\* FEM, Linear Algebra, Eigenvalue Problems

\## ▶️ How to Run



1\. Clone the repository:

```bash

git clone https://github.com/your-username/fem_solver_python.git

cd fem_solver_python

```



2\. Install dependencies

```bash

pip install -r requirements.txt

```


3\. Modify parameters of the beam in solver.py

Open solver.py in ATOM.io

Edit the beam parameters


4\. Run the solver

python solver.py



📈 Output

Natural frequencies of the system

Mode shapes. # Note this solver works only for torsion mode shapes.
